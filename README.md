Programa de Heladerías.

Este programa permite consultar distintas heladerías y dentro de ellas crear perdidos. Al crear el pedido, se pueden añadir distintas comidas, editarlas y/o eliminarlas. Al final del pedido se pueden ver los detalles de éste y el precio total a pagar.
